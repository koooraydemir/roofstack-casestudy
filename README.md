# Restaurant Order Management System

A modern restaurant order management dashboard built with Nuxt 4 and Vue 3.

## Tech Stack

- Nuxt 4
- Vue 3
- TypeScript
- Tailwind CSS
- Pinia
- Firebase Firestore
- Heroicons

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Real-time order tracking
- Multi-status order management
- Responsive design
- Product management
- Order creation and updates
